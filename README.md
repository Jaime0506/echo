## 📖 The Story Behind Echo
For a long time, Linux users have enjoyed **[Blanket](https://github.com/rafaelmardojai/blanket)**, a fantastic ambient sound mixer. However, upon transitioning to Windows and macOS, I found myself deeply missing this tool. While there is a macOS alternative named *[Blankie](https://github.com/codybrom/Blankie)*, it carries over the exact same visual language of Blanket, which didn't quite fit my aesthetic preferences for a modern desktop.

This gap inspired the creation of **Echo** (a temporary codename). The goal? To build a cross-platform alternative that offers the same relaxing, distraction-free auditory environment, but wrapped in a stunning, highly polished, and distinctly **modern design**. 

## 🚀 Current Status (MVP)
Echo is actively under development. It is currently in its **Minimum Viable Product (MVP)** stage, progressing through the following high-level phases:

- 🎨 **Phase 1: Visual Identity & Foundation (Current)**
  Establishing a clean, modern interface, structuring the project, and ensuring a native, seamless window experience across macOS and Windows.
- ⚙️ **Phase 2: Core Audio Engine**
  Integrating the audio player through the Rust backend, adding essential high-quality ambient tracks (Rain, Coffee Shop, White Noise), and linking the interface controls.
- ✨ **Phase 3: Polish & Expansion**
  Adding custom soundscapes, user presets, and finalizing the cross-platform executables for release.

## 🛠️ Tech Stack
Echo leverages modern web technologies packaged into a lightweight, blazing-fast desktop executable:
* **Core Desktop Framework:** Tauri v2 & Rust
* **Frontend UI:** React 19
* **Styling & Animations:** Tailwind CSS v4, Framer Motion, Radix UI
